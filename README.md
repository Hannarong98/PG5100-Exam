## About this application 

Its a small hearthstone card collector :)

## to run this application

***Prerequisite***

* JDK 11

***Instruction***

* Do ``mvn clean install``

* Under the frontend module in test folder run the ``LocalApplicationRunner``

* Connect to localhost:8080
